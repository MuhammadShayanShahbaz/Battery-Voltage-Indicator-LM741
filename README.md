# Battery-Voltage-Indicator-LM741
An analog battery voltage state indicator built using LM741 operational amplifiers and discrete components to provide visual LED feedback on battery health.  A Linear Circuit Analysis project featuring a battery voltage monitor that uses op-amps in comparator mode to trigger color-coded LED state alerts.
Author: Muhammad Shayan Shahbaz
Software: Proteus
Hardware: 
1) Operational amplifiers (IC 741)
2) Light-emitting diodes (LEDs)
3) Potentiometers (Variable resistors)
4) Fixed resistors
5) Batteries
